Features

  • Powerfull Code Editor

    NINJA-IDE provides a complete code editor with highlighting for several languages, code completion, code assistant for: imports, navigation, etc. Also it is possible to extend the editor functionality using plugins. Learn More

  • Display Manager

    Provides support to use an embedded Python Console into the IDE, and load several plugin's widgets for multi-purposes. Learn More

  • Project Managment

    NINJA-IDE allows to manage Python projects automatically, saving descriptive information about them and letting the user perform file managment related task in the IDE itself. Learn More

  • Run Projects and Files

    With NINJA-IDE it is possible to run Python projects or any file opened in the Editor with just one click. It is also possible to render an HTML file loaded in the Editor in the IDE to see how the changes would look like in a browser. Learn More

  • Highly extensible

    You can create a plugin for several purposes and it can be integrated completely with the IDE, increasing the available functionality to improve assistance for project development. Learn More
